# Nodejs commonjs ejs server

Only Nodejs and EJS application :zany_face:

Start the application

```shell
npm run start
```

Start dev mode without the need of dotenv in the dependencies

```shell
npm run start:dev
```

Watch dev mode

```shell
npm run watch:dev
```
